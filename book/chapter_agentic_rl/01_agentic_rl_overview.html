    • xпјҡжЁЎеһӢзҡ„иҫ“е…ҘпјҲеҰӮз”ЁжҲ·жҢҮд»ӨгҖҒжҸҗзӨәиҜҚпјүгҖӮ
    • yвҲ—пјҡеҜ№еә”зҡ„дәәзұ»ж ҮжіЁзҡ„зҗҶжғіиҫ“еҮәпјҲеҚі"ж ҮеҮҶзӯ”жЎҲ"пјүгҖӮ
    • DSFTвҖӢпјҡж”¶йӣҶеҘҪзҡ„жҢҮд»Ө - зӯ”жЎҲеҜ№ж•°жҚ®йӣҶгҖӮ
  • DSFTвҖӢ={(x(i),yвҲ—(i))}i=1NвҖӢпјҡй«ҳиҙЁйҮҸзҡ„ Agent дәӨдә’иҪЁиҝ№ж•°жҚ®йӣҶпјҢжҜҸжқЎж ·жң¬еҢ…еҗ«иҫ“е…ҘдёҠдёӢж–Ү xпјҲзі»з»ҹжҸҗзӨә + з”ЁжҲ·й—®йўҳпјүе’Ңдё“е®¶зӨәиҢғиҫ“еҮә yвҲ—пјҲеҗ«жҺЁзҗҶиҝҮзЁӢе’Ңе·Ҙе…·и°ғз”Ёпјү
  • logПҖОёвҖӢ(yвҲ—вҲЈx)пјҡжЁЎеһӢеңЁз»ҷе®ҡиҫ“е…Ҙ x зҡ„жқЎд»¶дёӢпјҢз”ҹжҲҗдё“е®¶зӨәиҢғеәҸеҲ— yвҲ— зҡ„еҜ№ж•°жҰӮзҺҮгҖӮиҝҷдёӘеҖји¶ҠеӨ§пјҲи¶ҠжҺҘиҝ‘ 0пјүпјҢиҜҙжҳҺжЁЎеһӢи®Өдёәдё“е®¶зӨәиҢғжҳҜ"еҗҲзҗҶзҡ„иҫ“еҮә"пјӣеҠ иҙҹеҸ·еҗҺеҸҳдёәжҚҹеӨұпјҢжңҖе°ҸеҢ–жҚҹеӨұеҚіжңҖеӨ§еҢ–еҜ№ж•°жҰӮзҺҮ
  • е®һйҷ…и®Ўз®—ж—¶пјҢlogПҖОёвҖӢ(yвҲ—вҲЈx) жҢүиҮӘеӣһеҪ’еҲҶи§Јеұ•ејҖдёәйҖҗ token зҡ„еҜ№ж•°жҰӮзҺҮд№Ӣе’ҢпјҡвҲ‘t=1вҲЈyвҲ—вҲЈвҖӢlogПҖОёвҖӢ(ytвҲ—вҖӢвҲЈx,y<tвҲ—вҖӢ)пјҢеҚіжҜҸдёӘ token зҡ„з”ҹжҲҗжҰӮзҺҮйғҪд»ҘеүҚйқўжүҖжңү token дёәжқЎд»¶

зӣҙи§үзҗҶи§ЈпјҡSFT зҡ„жң¬иҙЁжҳҜ"жЁЎд»ҝеӯҰд№ "вҖ”вҖ”з»ҷжЁЎеһӢзңӢеӨ§йҮҸдё“е®¶зӨәиҢғпјҢи®©жЁЎеһӢеӯҰдјҡ"еңЁиҝҷз§Қиҫ“е…ҘдёӢпјҢдё“е®¶дјҡиҫ“еҮәд»Җд№Ҳ"гҖӮиҝҷдёӘйҳ¶ж®өзұ»дјјдәҺ"дёҙеё–з»ғеӯ—"вҖ”вҖ”е…ҲжЁЎд»ҝжӯЈзЎ®зҡ„иЎҢдёәжЁЎејҸпјҢе»әз«Ӣж јејҸи§„иҢғе’Ңеҹәжң¬иғҪеҠӣгҖӮ

йҳ¶ж®өдәҢпјҡRLпјҲејәеҢ–еӯҰд№ пјүвҖ”вҖ” зӯ–з•ҘдјҳеҢ–

ж ёеҝғзӣ®ж ҮпјҡеңЁ ПҖSFTвҖӢ зҡ„еҹәзЎҖдёҠпјҢйҖҡиҝҮеҘ–еҠұдҝЎеҸ·еј•еҜјзӯ–з•Ҙеҗ‘ ПҖвҲ— йҖјиҝ‘пјҢзӘҒз ҙ SFT ж•°жҚ®зҡ„иҙЁйҮҸдёҠз•ҢгҖӮ

RL йҳ¶ж®өзҡ„и®ӯз»ғзӣ®ж ҮжҳҜжңҖеӨ§еҢ–жңҹжңӣеҘ–еҠұпјҢеҗҢж—¶йҖҡиҝҮ KL ж•ЈеәҰзәҰжқҹйҳІжӯўзӯ–з•ҘеҒҸзҰ»иҝҮиҝңпјҡ

LRLвҖӢ(Оё)=вҲ’EП„вҲјПҖОёвҖӢвҖӢ[R(П„)]+ОІвӢ…DKLвҖӢ(ПҖОёвҖӢвҲҘПҖSFTвҖӢ)

йҖҗйЎ№и§ЈиҜ»пјҡ

  • вҲ’EП„вҲјПҖОёвҖӢвҖӢ[R(П„)]пјҡзӯ–з•ҘжҚҹеӨұйЎ№вҖ”вҖ”жңҖеӨ§еҢ–еҪ“еүҚзӯ–з•Ҙ ПҖОёвҖӢ йҮҮж ·еҲ°зҡ„иҪЁиҝ№зҡ„жңҹжңӣеҘ–еҠұгҖӮиҙҹеҸ·жҳҜе°ҶжңҖеӨ§еҢ–иҪ¬дёәжңҖе°ҸеҢ–пјҲжўҜеәҰдёӢйҷҚжғҜдҫӢпјү
  • ОІвӢ…DKLвҖӢ(ПҖОёвҖӢвҲҘПҖSFTвҖӢ)пјҡKL ж•ЈеәҰжғ©зҪҡйЎ№вҖ”вҖ”DKLвҖӢ иЎЎйҮҸеҪ“еүҚзӯ–з•Ҙ ПҖОёвҖӢ дёҺ SFT еҲқе§Ӣзӯ–з•Ҙ ПҖSFTвҖӢ д№Ӣй—ҙзҡ„"еҲҶеёғи·қзҰ»"гҖӮеҪ“дёӨдёӘеҲҶеёғе®Ңе…ЁзӣёеҗҢж—¶ DKLвҖӢ=0пјӣе·®ејӮи¶ҠеӨ§пјҢDKLвҖӢ и¶ҠеӨ§гҖӮзі»ж•° ОІ жҺ§еҲ¶жғ©зҪҡејәеәҰпјҡОІ и¶ҠеӨ§пјҢзӯ–з•Ҙи¶Ҡдҝқе®ҲпјҲдёҚж•ўеҒҸзҰ» SFT жЁЎеһӢпјүпјӣОІ и¶Ҡе°ҸпјҢзӯ–з•Ҙи¶ҠжҝҖиҝӣпјҲеҸҜиғҪдә§з”ҹеҘ–еҠұй»‘е®ўиЎҢдёәпјүгҖӮ

    рҹ’Ў д»Җд№ҲжҳҜ KL ж•ЈеәҰпјҹ еҰӮжһңдҪ еҜ№ KL ж•ЈеәҰиҝҳдёҚзҶҹжӮүпјҢжҺЁиҚҗйҳ…иҜ»жң¬д№Ұзҡ„ йҷ„еҪ• EпјҡKL ж•ЈеәҰиҜҰи§ЈпјҢе…¶дёӯеҢ…еҗ«д»Һзӣҙи§үзҗҶи§ЈеҲ°ж•°еӯҰе®ҡд№үзҡ„е®Ңж•ҙз§‘жҷ®гҖӮз®ҖеҚ•жқҘиҜҙпјҢDKLвҖӢ(PвҲҘQ) еӣһзӯ”зҡ„й—®йўҳжҳҜпјҡ"еҰӮжһңзңҹе®һеҲҶеёғжҳҜ PпјҢз”ЁеҲҶеёғ Q жқҘиҝ‘дјје®ғпјҢе№іеқҮдјҡжҚҹеӨұеӨҡе°‘дҝЎжҒҜпјҹ"

  • дёӨйЎ№зҡ„еҚҡејҲпјҡзӯ–з•ҘжҚҹеӨұйЎ№йј“еҠұжЁЎеһӢ"еӨ§иғҶжҺўзҙў"д»ҘиҺ·еҫ—жӣҙй«ҳеҘ–еҠұпјҢKL жғ©зҪҡйЎ№зәҰжқҹжЁЎеһӢ"дёҚиҰҒиө°еӨӘиҝң"д»ҘдҝқжҢҒиҜӯиЁҖиҙЁйҮҸгҖӮОІ жҳҜиҝҷдёӨз§ҚеҠӣйҮҸзҡ„е№іиЎЎзӮ№

SFT еҸӘиғҪи®©жЁЎеһӢиҫҫеҲ°и®ӯз»ғж•°жҚ®зҡ„ж°ҙе№ідёҠз•ҢгҖӮиҖҢ RL йҳ¶ж®өйҖҡиҝҮеҘ–еҠұеҮҪж•°е‘ҠиҜүжЁЎеһӢ"д»Җд№ҲжҳҜеҘҪзҡ„з»“жһң"пјҢи®©жЁЎеһӢиҮӘдё»жҺўзҙўеҮәи¶…и¶ҠзӨәиҢғж•°жҚ®зҡ„и§ЈеҶіи·Ҝеҫ„вҖ”вҖ”иҝҷжӯЈжҳҜ DeepSeek-R1 [3] иғҪеӨҹж¶ҢзҺ°еҮә"иҮӘжҲ‘еҸҚжҖқ"е’Ң"й•ҝй“ҫжҺЁзҗҶ"иғҪеҠӣзҡ„е…ій”®жңәеҲ¶гҖӮ

SFT йҳ¶ж®өеӯҰеҲ°пјҡ  "зңӢеҲ°ж•°еӯҰйўҳ вҶ’ и°ғз”Ёи®Ўз®—еҷЁ"
