# 第五部分：生产化篇

> 🏭 将 Agent 从实验室带到生产环境，涵盖评估、安全与部署的全流程。

本部分包含 3 章，帮助你将 Agent 打造为可靠的生产级服务。

| 章节 | 内容 | 核心收获 |
|------|------|---------|
| **第16章** 评估与优化 | 基准测试、Prompt 调优、成本控制、可观测性 | 量化和提升 Agent 表现 |
| **第17章** 安全与可靠性 | Prompt 注入防御、幻觉治理、权限隔离 | 构建安全可信的 Agent |
| **第18章** 部署与生产化 | API 服务化、容器化、流式响应 | 上线生产级 Agent 服务 |
